문제1> '주문' 테이블에 '할인율'이라는 열을 추가하시오 

  ★  새로운 열을 추가할때 사용
     ALTER TABLE  테이블명
     ADD  열이름 타입선언;

     ALTER TABLE  주문
     ADD  할인율 타입선언 VARCHAR(10);

문제2> '주문' 테이블에 '할인율'의 이름을 'DC'로 바꾸시오.
 ★  ALTER TABLE  테이블명
    RENAME COLUMN  기존 열이름 TO 새로운 이름;

    ALTER TABLE  주문
    RENAME COLUMN  할인율 TO DC;

문제3> '주문' 테이블에 'DC'의 자료형을 NUMBER(5) 로 바꾸시오.
 ★  열의 자료형을 변경하는 MODIFY
 ALTER TABLE 이름
 MODIFY 열이름 자료형 ; 

 ALTER TABLE  주문
 MODIFY DC NUMBER(5) ;
